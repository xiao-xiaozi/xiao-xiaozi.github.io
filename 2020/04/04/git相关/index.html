git常用命令

记录一些git常用命令、便于需要时可以查看。

初始化仓库

在项目文件夹下执行git init即可将该文件夹变成git可以管理的仓库。执行该命令后当前目录会出现一个.git文件夹。这个是git用来跟踪版本管理库的。一旦删除，也就遗失了git管理。

向git仓库添加文件

一个文件进入到git仓库需要经过两步操作，首先将文件添加进暂存区，再提交到git仓库。
第一步、git add 执行该命令即将文件添加到暂存区

第二步、git commit 将暂存区的文件，提交到当前分支。

git commit带上-m参数后面输入的是本次提交的说明。也可以不带-m参数，直接打开编辑器进行提交信息的输入。

查看仓库状态

在进行文件操作的过程中，随时可以使用git status查看当前仓库的状态。文件的改动、新增、是否添加进缓存区。提交(commit)之后查看状态则会提示没有进行过改动。
